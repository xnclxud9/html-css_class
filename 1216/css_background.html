css속상-배경색 p.293

background속성은 사용자가 선택한 요소에 배경색, 배경이미지를 삽입하기 위한 속성이다.

배경속성의 종류

  1. background-color : 배경색(단일색, 그라디언트)
  2. background-image : 배경이미지
  3. background-repeat : 배경이미지 반복설정하기
  4. background-size : 배경이미지 크기
  5. background-position : 배경이미지 위치 변경
  6. background-origin : 배경이미지 기준위치 지정
  7. background-attatchment : 배경이미지 고정하기
  8. 멀티백그라운드 : css3에서 추가된 속성으로써 여러 이미지를 한꺼번에 삽입가능

1. 배경색(background-color)

배경색상을 넣기 위한 속성으로써 종류에는 '색상명', 'HEX코드', 'RGB', 'RGBA', 'HSL', 'HSLA'종류가 있다.

a는 alpha(투명도)의 약자로써 %를 소수점으로 환산하여 입력한다.

2. 그라디언트(gradient) p.305
