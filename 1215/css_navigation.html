내비게이션 - 사용자가 원하는 정보를 찾기 위한 기본적인 수단으로써 내비게이션이라고 하며, 내비게이션의 종류로는 '메인메뉴', '서브메뉴', 검색, 로그인, 사이드메뉴, 드롭다운, 탭메뉴, 사용자의 현재 위치를 알려주는 '로케이션'메뉴 등이 있다.

내비게이션은 주로 목록을 나열하기 위해 목록태그(ul, li)태그를 사용하며, a요소로 하이퍼링크를 설정한다.

gnb : Global Navigation Bar의 약자로써 보통 메인메뉴를 작성할 때 사용하는 명칭

lnB : Local Navigation Bar의 약자로써 메인메뉴외의 서브메뉴나 콘텐츠 영역에서 사용되는 메뉴를 작성하고자 할 때 사용하는 명칭이다.
