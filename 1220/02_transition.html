transition(전환)속성은 사용자가 선택한 html요소에 마우스를 올리거나 클릭을 하였을 때 첫장면에서 마지막 장면으로 자연스럽게 넘어가는 효과를 구현하고자 할 때 사용한다.

  • 모바일 토글버튼을 클릭시 부드럽게 나오는 내비게이션,
  • pc화면에서 메인 내비게이션 마우스 오버시 나오는 서브메뉴
  • 메인화면에 좌, 우 버튼 클릭시 슬라이드 되는 이미지
  • 부모페이지에서 자식요소를 띄우고, '닫기'누르면 부드럽게 사라지는 애니메이션 효과
  • 갤러리 이미지에 마우스 오버시 캡션나오게 하기
  • 기타....... UI 콘텐츠 요소에 다양하게 응용, 적용가능.
