02_css스타일 적용방법 p.173

스타일 시트는 상황에 따라 적용방법이 3가지로 나눈다.

  1. 내부스타일 - 현재문서의 상단 header에 style선언하고 /style 안에서 작성되어 현재문서의 내용에만 서식을 적용하기 위한 방법이다.
  2. 외부스타일 - 별도의 style문서를 css파일로 저장하여 html문서의 head안에 link태그로 연결하여 서식을 적용하는 방법이다. 여러개의 웹페이지에 같은 서식을 적용하고자 할 때 사용.
  3. 인라인스타일 - 태그안에 style속성을 사용하여 직접 적용하는 방식으로써 자주 사용되지는 않지만 테스트용으로 사용한다.
css에서 주석은 /* 한줄주석 */     

    /*
    여러줄 주석
    여러줄 주석
    */

    을 사용한다.
