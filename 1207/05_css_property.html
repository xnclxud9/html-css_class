05_css_property(속성) p.194

css언어는 '선택자{속성:값;}'으로 작성된다. 여기서 속성은 사용자가 선택한 요소에 다양한 서식을 적용하기 위한 종류에 해당된다.

css속성의 종류

  • 글자관련 : font-family(글자모양), font-size(글자크기), color(글자색상), text-decoration(밑줄, 줄없음, 윗줄), text-transform(영문 대, 소문자 전환), font-style(글자 기울임, 보통)
  • 문단관련 : text-align(정렬), text-indent(들여쓰기), line-height(행간)
  • 크기관련 : width(가로크기), height(세로크기)
  • 여백관련 : margin(바깥), padding(안쪽)
  • 테두리관련 : border(테두리 선)
  • 배경관련 : background-color, background-image, background-size, background-position......
  • 위치관련 : position, float, transform:translate(가로위치,세로위치)
  • 레이아웃관련 : flex, grid, display
  • 애니메이션 관련 : animation, transition

1. 글자, 문단관련 속성

font-family : 글자모양을 지정하고자 할 때 사용하는 속성, 한글폰트를 지정하고자 할 때는 "인용부호를 사용해야 하며, 영문글꼴일 경우는 공백이 있는 글꼴일 때만 인용부호를 붙인다.

font-size : 글자크기를 지정하는 속성으로써 px, %, em, rem단위가 있다. 기본값은 px단위를 많이 사용. 브라우저의 기본값은 16px임.

font-style : 글자 스타일을 지정할때 사용하는 속성이며, normal(보통), iltalic(기울게), oblique, inherit등의 속성값이 있다.

font-weight : 글자를 굵게, 얇게, 보통으로 변경하기 위한 속성. bold, bolder, normal, lighter, 100~900까지 속성값이 있다.

font-variant : 영문자를 작은 대문자로 변환해주는 속성.

text-transform : 영문 대소문자를 대문자로 전환, 대문자를 소문자로, 단어의 첫글자를 대문자로 전환시켜주는 속성.

text-decoration : 글자에 밑줄, 윗줄, 줄없음을 지정하는 속성

text-align : 문단정렬속성으로써 left, right, center, justify
