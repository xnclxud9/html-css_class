13_form_add(html5에서 추가된 폼양식 태그와 속성) p.130

HTML5에서 추가된 폼태그와 속성들은 현재 사용하고 있는 브라우저(크롬, MS EDGE, 파이어폭스, 오페라, 사파리, 웨일, 스윙.....)에서 버전별 지원을 하기 때문에 지원상황을 확인하고 사용해야 한다. http://caniuse.com(브라우저 버전별 태그 지원 유무 확인 사이트)

1. 폼양식을 그룹지정하기 위한 요소 - feldset, legend

필수입력정보

선택정보

placeholder : 사용자가 입력할 내용을 미리 알려주어 오류가 나지 않도록 방지해줌. 웹접근성 측면에서 도움이 많이 됨.

search : 사용자가 입력양식에 키워드를 입력하여 검색할 수 있도록 하고자 할 때 사용하는 양식으로써 단어를 입력하면 오른쪽에 'x'버튼이 자동으로 뜨며 클릭하면 글 입력내용이 삭제되는 기능이 있다.

email : 이메일 주소를 입력하기 위한 박스로써 입력양식이 맞지 않을 경우 에러가 난다.

url : 사용자가 도메인 주소를 입력할 수 있도록 하는 양식이며, 주소형식이 맞지 않을 경우 에러메세지가 나온다.

number : 숫자를 입력하기 위한 양식으로써 직접입력하거나 버튼을 클릭하여 '증가', '감소'가 가능하다.

수량 선택 :

date : 년, 월, 일을 선택하기 위한 요소로써 클릭시 달력창이 활성화 됨.

날짜 선택 :

color : 사용자가 원하는 색상을 고르기 위한 양식

progress : 진행 정도를 나타내는 바를 만드는 태그로써 max, value속성을 사용합니다.
