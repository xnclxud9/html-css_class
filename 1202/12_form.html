12_form(폼) p.122

폼태그란? 사용자가 입력한 정보(id, password.....)를 서버로 전송하는 역할을 하며, 폼태그를 활용한 양식의 종류에는 로그인, 회원가입, 검색, 예약 폼 등 다양한 양식, 서식을 만들 수 있다.

양식을 만들때는 form태그로 묶어서 작성하며, name(폼이름), method(전송방법), action(받을 서버주소) 속성이 들어간다.

폼태그의 종류

  • form
  • fieldset, legend
  • input
  • select, option
  • textarea

폼태그 사용

1. 기본 폼태그

1) txt박스 : 한글, 영문자를 입력하고자 할 때 사용하는 양식이며, 가장 많이 사용되는 양식이다.

2) password박스 : 패스워드를 입력할 때 사용하는 서식이며 글자가 보이지 않도록 '기호'로 처리된다.

3) 라디오 버튼 : 여러개의 항목중에서 하나만 선택이 가능한 양식으로써 name속성을 사용해야 하나만 사용이 가능하다.

성별선택 :

라디오버튼, 체크박스는 사용자가 텍스트를 클릭하여 선택할 수 있도록 id값으로 설정한다. 웹접근성 측면에서 pc, tablet, mobile사용자가 편하고 쉽게 선택할 수 있는 하나의 수단이 됨.

4) 체크박스 : 여러개의 항목들 중에서 사용자가 원하는 항목만 선택할 수 있는 양식

나의 취미 :

5) 선택목록 : 사용자가 목록중에서 원하는 항목을 선택하는 양식으로 쇼핑몰 옵션선택, 전화번호 통신사 선택, 예매인원 선택, 언어설정, 패밀리사이트 등의 양식을 만들고자 할 때 사용

6) 다중텍스트 박스 : 한줄 입력이 아닌 여러줄의 내용을 입력하고자 할 때 사용

7) 파일첨부 : 게시판이나 이메일 양식에서 상대방에게 파일을 첨부하여 보내기 위한 양식

파일첨부하기 :

8) 버튼 : 전송하기, 취소하기, 일반버튼, 이미지버튼
