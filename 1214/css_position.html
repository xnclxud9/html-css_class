css_position(요소의 배치방법) p.280

포지션(position) 속성이란? 레이아웃 작업시 사용자가 부모요소 박스안에서 원하는 콘텐츠를 선택하여 배치하기 위한 속성

  • static - 요소가 가지고 있는 기본 위치값
  • absolute - 절대배치
  • relative - 상대배치
  • fixed - 고정배치
  • inherit - 부모로부터 물려받음(상속)

위 포지션속성을 사용시 left, rifgt, top, bottom 위치값을 설정할 수 있다.

z-index속성은 먼저 작성된 태그가 z-index값이 0이며, 나중에 작성된 태그 순서대로 번호가 자동으로 매겨진다. z-index속성을 사용하여 위/아래 순서도 변경이 가능하다.

1. 절대배치(absolute)

2. 상대배치(relative)

3. 고정배치(fixed)
